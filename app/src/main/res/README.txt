Rock, Paper, Scissors 0.8 - This app allows the user to play the game
of Rock, Paper, Scissors against a computer. The user clicks the radio
button represent their choice of hand throw. The computer will then
randomly choice a hand throw. The user will receive a point if they beat
the computer (Rock > Scissors, Scissors > Paper, Paper > Rock) and will
lose a point if they lose to the computer. No points are assigned for ties.
